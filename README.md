# نظام التسعير الذكي - مهووس v16.0

نظام متكامل لتسعير العطور بالذكاء الصناعي.

## المميزات
- محرك مطابقة ذكي (RapidFuzz)
- AI مباشر (Gemini + OpenRouter)
- مقارنة بصرية بالألوان
- معالجة في الخلفية
- استثناء العينات فقط
- أتمتة Make.com
- 12 قسم متكامل

## التشغيل
```bash
pip install -r requirements.txt
streamlit run app.py
```
